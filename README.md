# TRACKS

## Installing
From terminal run:
```sh
npm install
```

Installation of less and less watcher:
```sh
npm install -g less
npm install -g less-watch-compiler
```

## Bulding
From terminal run:
```sh
rollup -c
```

or with watcher:
```sh
rollup -c -w
```

Starting less watcher:
```sh
less-watch-compiler styling build main.less
```
